This is a simple c++ solution for the infamous interview question "Fizzbuzz"

Run Project:
Visual Studio 2017: Open the project (double click Fizzbuzz.sln). Once loaded, press the function key 'F5' or press the button that says "Local Windows Debugger".
Command Line (g++): g++ main.cpp Fizzbuzz.cpp -o Fizzbuzz; ./Fizzbuzz

Versions: 
g++ Version: g++.exe (x86_64-posix-seh-rev1, Built by MinGW-W64 project) 7.1.0
Visual Studio 2017
